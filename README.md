# Project Brief

## Project
Herbal Wellness Tea Brand Site

## Goal
Build a production-ready standalone product website for wellness tea.
The site should introduce products, educate users through articles about herbal ingredients,
and integrate the existing AI web-ui as a guided tea recommendation tool.

## Audience
- Chinese-speaking and English-speaking users
- Users interested in herbal wellness tea
- Users who need guided product recommendations

## Core Features
- Product catalog
- Product detail pages
- Ingredient knowledge library
- Articles / blog
- AI tea guide entry
- Shopping cart
- Mobile-first responsive design
- SEO-ready architecture
- Bilingual support

## Existing App
There is an existing web-ui in the current project that should be integrated as a helper tool.
Preferred integration path: /ai-guide or embedded section in site. Details refer to integration.md
